# 我的测试项目合计

## 项目一： 计算机自动化测试
** 技术栈：** Python, pytest

** 功能：** 
- 实现基本计算器（加减乘除）
- 使用pytest编写自动化测试
- 测试覆盖率100%

** 测试用例：** 5个

** 运行方式：**

## 项目二： 学生成绩管理系统
** 技术栈：** Python, pytest

** 功能：** 
- 实现学生成绩的管理（增删改查、成绩的平均值）
- 使用pytest编写自动化测试
- 对学生信息的增删改查及数字信息的等价类、边界值进行测试用例的设计
- 使用pytest.raises验证测试中的代码是否正确抛出了异常
- 测试覆盖率100%

** 测试用例：** 20个

** 运行方式：**
 pytest TestChapter/Test_StudentManager.py
